## Why I contribute

Open source expands the shared technical foundation. Making advanced models easier to inspect, reproduce, and use lets other engineers learn from the work, improve it, and create outcomes the original contributor could not anticipate. I contribute where a research artifact needs the engineering work required to become dependable public infrastructure.
