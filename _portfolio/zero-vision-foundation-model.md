ZERO is a multi-modal prompt-based visual grounding project developed at SuperbAI. It supports text and visual prompts for object detection, with a focus on production-ready deployment across diverse industrial domains.

Highlights:

* Built a prompt-driven object detection system for text and semantic visual prompts.
* Prepared large-scale in-house image data and model-assisted verification workflows.
* Applied domain-specific fine-tuning strategy for CVPR 2025 Foundational Few-Shot Object Detection work.
