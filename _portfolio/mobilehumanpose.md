MobileHumanPose is a mobile-friendly model for real-time 3D human pose estimation from a single RGB image.

Highlights:

* Designed an efficient 3D human pose estimation model for low-compute mobile devices.
* Used a modified MobileNetV2 backbone, parametric activation, and U-Net-inspired skip concatenation.
* Reduced model size compared with ResNet-50 based models and improved CPU inference latency on Galaxy S20.
